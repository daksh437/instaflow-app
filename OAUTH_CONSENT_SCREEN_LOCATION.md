# 📍 OAuth Consent Screen Kaise Find Karein

## 🎯 Current Location
Aap abhi **"OAuth Overview"** page pe ho. OAuth consent screen alag jagah hai.

## 🔍 Method 1: Left Sidebar Se (Easiest)

1. **Left sidebar** mein dekho - "Audience" ya "Settings" pe click karo
2. Ya phir **top pe search box** mein type karo: **"OAuth consent screen"**
3. Result mein **"OAuth consent screen"** pe click karo

## 🔍 Method 2: APIs & Services Section Se

1. **Top left** pe **hamburger menu (☰)** click karo
2. **"APIs & Services"** pe hover karo
3. Dropdown mein **"OAuth consent screen"** dikhega
4. Click karo

## 🔍 Method 3: Direct URL Se

Browser mein ye URL paste karo:
```
https://console.cloud.google.com/apis/credentials/consent?project=insta-flow-7d1a7
```

## 📋 Agar Phir Bhi Nahi Mile

### Alternative: Credentials Section Se

1. **Left sidebar** mein **"Clients"** pe click karo
2. Ya **top search** mein type karo: **"Credentials"**
3. **"APIs & Services" → "Credentials"** pe jao
4. Wahan se bhi OAuth consent screen access kar sakte ho

## 🎯 Exact Steps (Screenshot Ke Basis)

Aapke screenshot mein:
1. **Left sidebar** mein **"Audience"** pe click karo
   - Ya **"Settings"** pe click karo
   - Wahan OAuth consent screen ka option hoga

2. Ya **top search bar** mein type karo:
   ```
   OAuth consent screen
   ```
   - Result mein click karo

3. Ya **hamburger menu (☰)** → **"APIs & Services"** → **"OAuth consent screen"**

## ✅ Expected Screen

Jab OAuth consent screen milega, wahan dikhega:
- **User Type** selection (External/Internal)
- **App information** form
- **Scopes** section
- **Test users** section

## 🆘 Agar Abhi Bhi Nahi Mile

1. **Top search bar** mein type karo: **"OAuth consent"**
2. Ya **hamburger menu** → **"APIs & Services"** → **"OAuth consent screen"**
3. Ya direct **"Clients"** section check karo

---

**Note:** Google Cloud Console UI kabhi-kabhi update hota hai. Agar exact location different hai to screenshot share karo, main exact location bata dunga!
