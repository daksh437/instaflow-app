# Insta Analyzer Flutter App

Flutter frontend for Instagram profile analysis.

## Setup

1. Install dependencies:
```bash
flutter pub get
```

2. Run the app:
```bash
flutter run
```

## Backend Connection

The app uses the production backend URL that works on all platforms:

- **All Platforms**: Uses `https://insta-flow-backend.onrender.com`
- The backend is deployed on Render and runs 24/7
- No local development server required

## Features

- Search Instagram profiles by username
- View profile information (avatar, name, followers, engagement rate)
- Browse top posts with thumbnails
- View likes distribution chart
- Scheduler feature (coming soon)

## Testing

Make sure the backend is running before using the app. See the root README for instructions.
